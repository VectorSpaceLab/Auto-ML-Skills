# @auto-ml-skills/disco-agent-core

Core agent-loop primitives used by DisCo.

This package is part of the local DisCo source tree. It is built as a
workspace dependency of `@auto-ml-skills/disco` and is not intended to be
used as a standalone user-facing tool in this repository.

## Development

```bash
npm run build
```

The root `src/package.json` builds this package before the DisCo CLI.
