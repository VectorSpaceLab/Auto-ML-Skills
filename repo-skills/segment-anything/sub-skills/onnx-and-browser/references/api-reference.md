# ONNX API Reference

## `SamOnnxModel`

`SamOnnxModel(model, return_single_mask, use_stability_score=False, return_extra_metrics=False)` wraps a loaded SAM model for tracing. It combines prompt embedding, mask decoding, and mask postprocessing into an ONNX-compatible module.

Do not call this class as a normal segmentation API unless you are exporting or diagnosing ONNX behavior. For ordinary prompted Python inference, use `SamPredictor` through the prompted-segmentation sub-skill.

## Forward Signature

The ONNX wrapper forward inputs are:

```text
image_embeddings, point_coords, point_labels, mask_input, has_mask_input, orig_im_size
```

The forward path:

1. Embeds points with SAM positional encoding and point label embeddings.
2. Embeds the previous mask if `has_mask_input` is true; otherwise uses the learned no-mask embedding.
3. Calls `model.mask_decoder.predict_masks` with the image embedding, dense positional encoding, sparse prompt embeddings, and dense prompt embeddings.
4. Optionally replaces predicted IoU scores with stability scores.
5. Optionally selects a single best mask.
6. Upscales masks from decoder resolution to the original image size.

## Inputs

- `image_embeddings`: generated by `SamPredictor.get_image_embedding()` after `set_image`; shape is usually `[1, 256, 64, 64]` for SAM's 1024 image encoder size.
- `point_coords`: resized prompt coordinates, shape `[1, num_points, 2]`; dynamic axis `num_points` is exported.
- `point_labels`: prompt labels, shape `[1, num_points]`; dynamic axis `num_points` is exported.
- `mask_input`: previous low-resolution mask logits, shape `[1, 1, 256, 256]`; zeros are valid for first-pass prompts.
- `has_mask_input`: shape `[1]`; `[0]` for no prior mask, `[1]` to refine using `mask_input`.
- `orig_im_size`: original image size as `[height, width]`, shape `[2]`.

## Outputs

Default outputs:

- `masks`: upscaled mask logits at original image size.
- `iou_predictions`: mask quality predictions, or stability scores when `use_stability_score` is enabled.
- `low_res_masks`: low-resolution logits suitable for iterative refinement as the next `mask_input`.

With `return_extra_metrics=True`, outputs are:

- `masks`
- `iou_predictions`
- `stability_scores`
- `areas`
- `low_res_masks`

Consumers that assume three outputs must be updated before enabling `--return-extra-metrics`.

## Export Script Behavior

The bundled export script loads a SAM checkpoint with `sam_model_registry[model_type]`, creates `SamOnnxModel`, builds dummy inputs, traces with `torch.onnx.export`, and validates the exported file with ONNXRuntime when available. Quantization uses dynamic uint8 weights through ONNXRuntime quantization.
