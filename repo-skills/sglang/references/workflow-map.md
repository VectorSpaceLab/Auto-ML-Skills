# SGLang Workflow Map

| Goal | Start here | Notes |
| --- | --- | --- |
| Launch or debug an SGLang server | `sub-skills/serving-runtime` | Owns CLI/server args, OpenAI-compatible endpoints, scheduler/cache behavior, parallelism, disaggregation, and runtime dependencies. |
| Write SGLang frontend programs | `sub-skills/frontend-programming` | Owns decorators, generation primitives, choices, role helpers, runtime endpoints, and frontend backend selection. |
| Measure performance | `sub-skills/benchmarking-profiling` | Owns benchmark scripts, request datasets, metrics, GPU profiling, and reproducibility. |
| Add model or kernel support | `sub-skills/model-kernel-extension` | Owns model configs, custom model integration, CUDA/JIT kernels, and related repo-local cookbook skills. |
| Change the repository | `sub-skills/repo-development` | Owns source layout, focused tests, CI conventions, docs/cookbook checks, and dependency caveats. |
